# Audit samples

This folder contains small anonymized sample payloads for repository audit and smoke tests.

They are not real Hypocampus captures and they are not extracted from faculty PDFs. Medical text is synthetic and intentionally generic. The goal is only to expose representative JSON shapes without publishing the private `data/` folder.

Contents:

- `captures/`: 3 sample captured questions.
- `annales/`: 1 sample annale with QRM, QRU and QROC examples.
- `exam-sessions/`: 3 sample completed sessions referencing the sample annale.

Do not add real user captures, real exam PDFs, API keys, or production `data/` files here.
